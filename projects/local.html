Downloading Games on Chromebooks

Administrators can block certain websites. A large amount of blocked websites are pages used to play video games or entertainment. There is a very easy workaround that allows you to play games hosted on your computer.

How to download games and run them:
1. Find a game on GitHub that only has code in these three formats: HTML, CSS, JS.
2. Download it from GitHub as a Zip file.
3. Go to the Zip file in your files and find the "Index.html" file inside of it.
4. Double click the html and it will open the game as a tab.

You may notice that the link is not normal, this is because you are running a file instead of going to a website.
